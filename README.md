# Talk-RTL
Repositório da Talk sobre React Testing Library

## Setup

### Clone o Repositório
```
git clone https://github.com/CaioMassayuki/Talk-RTL
```

### Entre na pasta do projeto
```
cd Talk-RTL/talk-rtl
```

### Instale as Dependencias
```
npm i
```

## Scripts

### Iniciar Projeto
```
npm start
```

### Rodar Testes
```
npm test
```

### Rodar Coverage
```
npm run test:ci
```
